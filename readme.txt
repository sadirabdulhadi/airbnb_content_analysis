#KEY DIRECTORIES

Census data extraction:
* Script to calculate the socio-economic metrics

Census data correlation:
* Script to build the cross-correlation matrix

Linear Regression:
* The code automatically runs all regressions (for all metrics, all listings...)

LIWC and Mydict metric calculation:
* These directories contain code which calculates scores for every review. We later take the median of these scores on the ward level.

Plot metric change and plots:
* These directories contain code to plot graphs
